## Results of KG processing

The results of KG processing contains the following data

* General informations

	- creation Data
	- measurement location
	- measurementsession/name
	- measurement datum
	- KG algorithm
	- KG algorithm version

* description of data table

	- list of column names
	- list of values for every column

* Table of data

### Output Format

filename :

date,dd-mm-yyy
loc,location
measurement, name
KG_alg,function Name

colnames,
