Description of the software

The aim of this program is to provide a way of selecting ranges where trains squeal or flang. Each case is denoted by a measure code m_0***_# (where # denotes the number of the microphone). You are kindly asked to select the ranges where you here a squealing noise or a flanging noise using both the audio - accessible with the media buttons :

and the associated plot - it is the sound level with fast weighing




Startup

Select a username (your initials for example)

Then select the number of cases you would like to treat. Note that it takes approximately 5 to 10 minutes per case. You do not have to analyse every case you chose, and you do no have to analyse them in order.



After you have answered these two questions, you can start analysing the cases you selected



ou can come back to this help page whenever you need, it opens on your default web browser. If you closed the tab, simply click on the INFO button on top of the program's page and this help will open again in your web browser.




To fulfil the analysis, please proceed as follow :

  1. Listen to the recorded signal using the media buttons in the toolbar

    • Warning: This step require a silent environment and an quality audio system.
      • Notebook speakers have poor quality.
      • Use hearphones or a better audio system to analyze the signal.
    • We look at two types of noise:
      • Kreischen: Squeal noise.
      • Zischen: Flanging noise.

  2. Select the intervals (ranges) where you hear a squealing/flanging noise

    1. Check that you have selected the right type of noise under Noise Type to select. (You can click both visible if you want to see the interval you selected for both noise type. Unclick it if you only want to see the noise type selected).

    2. Select the intervals by either:

      • Using left mouse click to span the interval (works with and without the audio playing)
      • Hold key s while the audio is playing to select the interval spanned by the orange bar.


    3. If necessary, you can remove :

      • an entire interval, by right-clicking on it;
      • an interval of your choice, by spanning the interval with right click or alternatively, by holding key d while spanning the interval with left mouse clicked.

    4. Select the quality of the sampling

      • good: noise events are good to recognize;
      • medium: it is possible to detect noise events;
      • bad: noise events almost impossible to detects.

    5. If you have not completed the selection for both type of noise, go back to 1.

    6. Save the intervals you have selected

      • To do this check that the saving path is appropriate (if it is None you'll be asked for one) :
      • Click the button Save (it will change color from violet to green)

    7. Go to a new case

      • To select the intervals for a new case (after having saved the current one), click on the combo box under Select case to analyze.

      • Notice that the cases that were saved are now green in the combo list:

    If you need to change the saving folder, click on the button Change folder under Saved data directory :

    Make sure you have analyzed and saved all the cases before quitting the application. A saved case has a green state (on the combo list and on the save button).
