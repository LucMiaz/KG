Description du programme

Ce programme est le premier pas dans l'élaboration d'un algorithme efficiant capable de détecter les bruits de flanc et de crissement produits par les roues des trains. Dans cette perspective, nous souhaitons rassembler un ensemble de séquences sélectionnées manuellement pendant lesquelles on entend un des deux bruit est présent. Nous avons sélectionné cent cas (i.e. un fichier audio avec quelques charactéristiques). Tous ces cas possèdent un code de mesure de la forme m_0***_# (où # désigne le numéro du microphone). Nous vous demandons d'effectuer une analyse de quelques un de ces cas. Vous êtes prié de sélectionner le domaine (les intervalles) où vous entendez un des deux bruits. Pour ce faire, il faudra utiliser les boutons play/pause/stop pour activer le fichier audio et s'aider du graphique associé (il représente le niveau sonore avec un filtre fast time weighting. Startup

  • Questions de départ :

    • Sélection d'un nom d'utilisateur (par exemple vos initiales)



    • Il faut ensuite sélectionner le nombre de cas que vous souhaitez étudier.

      • Chaque cas nécessite entre 5 et 10 minutes. Vous n'avez pas besoin d'analyser tous les cas qui vous seront proposés, ni de les analyser dans l'ordre.



  • Après avoir répondu à ces deux questions vous pourrez commencer l'analyse.

  • Vous pouvez retourner à cette page d'aide quand vous le souhaitez. Elle est ouverte dans votre navigateur web.

    • Si vous avez fermé l'onglet de la page d'aide, il vous suffit de cliquer le bouton INFO du programme pour que cette page s'ouvre à nouveau sur votre navigateur.





Veuillez procédez comme suit pour analyser les cas:

  1. Écoutez l'enregistement du signal sonore en utilisant les boutons Play/pause/stop dans la barre de menu ou en pressant sur la touche keyboard shortcutespace

    • Attention: Il vous faut réaliser cette analyse dans un environnement calme et avec un système d'écoute de bonne qualité.
      • En effet, les hauts-parleurs des ordinateurs portables sont généralement de qualité trop faible pour bien différentier les bruits.

      • Préférez l'utilisation d'écouteurs ou de bons hauts-parleurs.

    • Nous différentions deux types de bruit (voir Tableau des différentes fréquences):
      • Kreischen: Crissement.
        • C'est un bruit fortement tonal[1] (c'est-à-dire dominé par une seule fréquence et survenant dans les courbes serrées). (Thompson 2009 : 315)
      • Zischen: bruit de flanc/grincement.
        • [...] les bruits de flanc sonnent à l'oreille humaine comme un grincement ou un sifflement [...](Eadie 2005 : 1153)
        • En comparaison avec le crissement dû à un glissement latéral, les bruits de flanc ont une fréquence fondamentale beaucoup plus élevée, peuvent avoir un niveau plus bas et sont souvent intermittants. Ils peuvent également avoir une bande plus large. (Thompson 2009 : 316)

  2. Sélection des intervalles (domaines) où vous entendez un crissement/grincement

    1. Vérifiez que vous avez sélectionné le bon type de bruit dans la case Noise Type to select (Passez d'un type à l'autre avec la touche T).
      • Vous pouvez cliquer sur both visible si vous souhaitez afficher les intervalles que vous avez sélectionnés pour les deux types de bruits.
      • Désélectionner la même case pour n'afficher que le bruit courant.
      • La touche B vous permet de passer d'un affichage à l'autre.



    2. Pour sélectionner les intervalles vous avez les possibilités suivantes :
      • Maintenez le bouton gauche de la souris pour couvrir l'étendue de l'intervalle (fonctionne avec ou sans l'audio)
      • Maintenir la touche S lorsque l'audio est joué. Cela sélectionnera l'intervalle couvert par la barre orange.


    3. Si nécessaire, vous pouvez supprimer un intervalle ou un sous-intervalle :
      • Un intervalle entier, en cliquant l'intervalle voulu avec le bouton droit de la souris;
      • Un sous-intervalle quelconque, en couvrant la bande à supprimer avec le bouton droit de la souris maintenu;
      • Un intervalle couvert par la barre audio en maintenant la touche KD lorsque l'audio est joué.

  3. Choisissez la qualité de l'audio que vous venez d'écouter au moyen des trois boutons radio :

    • good: les bruits sont facilement reconnaissables; (TOUCHE 3)
    • medium: il est possible de différencier les bruits; (TOUCHE 2)
    • bad: les bruits ne sont pas discernables. (TOUCHE 1)

  4. Si vous avez not complété la sélection pour un seul des deux bruit, retournez au point 1 et effectuez les mêmes opérations pour le deuxième bruit.

  5. Sauvegardez les intervalles que vous avez choisis

    • Pour cela, vérifiez que le chemin de sauvegarde est approprié :

    • Cliquez sur le bouton Save (sa couleur passera au vert);
    • Alternativement pressez simultanément les touches CTRL + S.


  6. Passer à un nouveau cas

    • Après avoir sauvegarder vos intervalles, vous pouvez passer à l'analyse d'un autre cas en cliquant sur la boîte de choix Select case to analyze.
    • Alternativement les touches gauche ou haut (ARROW UP/LEFT) de votre clavier vous amènereont au cas précédent et les touches droite ou bas (ARROW RIGTH/DOWN) au cas suivant.


      • Notez que les cas que vous avez sauvegarder seront green dans la boîte de choix:


Raccourcis clavier

TOUCHE EFFET
CTRL+S Sauve le cas
CTRL+SPACE Arrête l'audio
SPACE Play/Pause
S (avec l'audio joué) Sélectionne la bande couverte par la barre
D (avec l'audio joué) Supprime la bande couverte par la barre
B Passe d'un affichage à l'autre (deux types ou un seul)
1, 2, 3 Change la qualité audio ('bad', 'medium', 'good')
Flèche haut ou gauche Passe au cas précédent de la liste
Flèche bas ou droite Passe au cas suivant de la liste
Z, K, T Passe à la sélection du type de bruit suivant
F Change le chemin de sauvegarde
CTRL + R Change la résolution (dialogMenu)
CTRL + U Mets à jour le canvas
